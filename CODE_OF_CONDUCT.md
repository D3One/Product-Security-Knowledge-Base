# Code of Conduct

This repository aims to support a professional, respectful, and high-signal community around Product Security, AppSec, DevSecOps, and adjacent engineering practices.

## Expected behavior
- Be respectful.
- Be specific.
- Be constructive.
- Assume good intent.
- Prefer practical feedback over noise.

## Unacceptable behavior
- harassment or abusive conduct;
- spam or low-effort promotion;
- personal attacks;
- disruptive behavior that lowers the signal for others.

## Scope
This code of conduct applies to issues, pull requests, linked discussion spaces, and related interactions around this repository.

## Contact
For sensitive concerns, please reach out privately through the author's preferred contact channels.
