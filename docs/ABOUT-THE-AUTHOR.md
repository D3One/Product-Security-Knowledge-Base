# About the Author

Ivan Piskunov is a cybersecurity practitioner focused on **Product Security, AppSec, DevSecOps, Cloud Security, Secure SDLC, and leadership-oriented security operating models**.

He has worked across **multiple software and fintech-related environments**, helping strengthen software security practices from both the engineering side and the security side. His trajectory includes hands-on work in **AppSec, DevSecOps, Security Champion responsibilities, platform and cloud security, and later Product Security-oriented positioning**.

Over time, that path evolved from technical implementation into a broader mission:

- systematize useful security knowledge;
- explain complex topics in a practical way;
- mentor younger engineers;
- help people build stronger hard skills;
- connect technical depth with career growth and real market value.

## Positioning

Ivan positions himself around **Product Security Director / VP-level scope**.

That means the work is not limited to tools or isolated testing activities. It is increasingly about:

- safer product and platform architecture;
- security enablement for engineering organizations;
- practical governance and metrics;
- risk reduction through delivery-integrated controls;
- scaling knowledge across teams;
- building a durable Product Security culture.

## Working style

This body of work reflects a few clear beliefs:

- Product Security should be practical, not theatrical.
- Engineers need usable guidance, not abstract slogans.
- Security programs should reduce friction where possible.
- Good security communication is part of the job.
- Great knowledge systems help teams move faster, not slower.

## Community contribution

A major part of the mission is contribution back to the industry.

That includes:

- publishing articles and educational material;
- documenting repeatable patterns and defensive practices;
- maintaining community-facing channels and repositories;
- helping newer engineers avoid low-signal advice;
- improving technical readiness for interviews and stronger offers;
- translating scattered experience into structured playbooks.
