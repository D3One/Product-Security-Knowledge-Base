# Contributing

Thank you for your interest.

At this stage, this repository functions primarily as a **presentation repository** for the broader Product Security Knowledge Base.

## Feedback currently welcomed

- broken links;
- readability issues;
- typos;
- navigation friction;
- high-value suggestions for structure and clarity.

## Contributors and co-authors

This project is open to strong improvement ideas and future editorial collaboration.

Meaningful contributors may later be recognized publicly as:

- contributors;
- editorial collaborators;
- co-authors for substantial work.

See the dedicated page: [Contributors and Co-Authors](docs/CONTRIBUTORS-AND-COAUTHORS.md)

## Preferred feedback style

Please keep feedback:

- concise;
- practical;
- specific;
- grounded in real use.

## Larger collaboration

If broader collaboration opens up later, this page can be expanded with contribution flow, standards, editorial rules, and recognition criteria.
